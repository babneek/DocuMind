# DocuMind AI - Frontend

Modern React 18 + TypeScript frontend for the DocuMind AI document intelligence platform.

## Features

- User authentication and authorization
- Document upload and management
- Intelligent document querying with AI
- Real-time search and retrieval
- Annotation and note-taking system
- Responsive design with Tailwind CSS

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
